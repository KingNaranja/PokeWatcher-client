# **PokeWatcher**



### Overview 

Pokewatcher is a web app where users can log sightings of unique legendary Pokemon.  User diary entries are recorded in a rails database.

### Technologies 

1. Javascript
2. HTML
3. CSS / SCSS
4. jQuery
5. Ajax
